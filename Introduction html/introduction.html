Please read the instruction below before using Role Management


  1. Select an application to start. If you have not yet create an application, please click "Add An Application" to continue.



  2. Once you selected an application, the panels on the left will expand and become your application-panel, role-panel, and assigned-panel.


  3. Each application includes different roles, and each role includes people and groups that are assigned.


  4. Let us know if you have any question or concern regarding Roles Management, thank you!
