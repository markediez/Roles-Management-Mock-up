Welcome to Roles Management

Please read the instruction below before using Roles Management


  1. Select an application to start. If you have not yet create an application, please click "Add An Application" to continue.



  2. Once you selected an application, the left panels will expand to become your application-panel, role-panel, and assigned-panel. The right panel is your control panel that allows you to make any changes to your applcations, roles, and assign people.


  3. Each application includes different roles, and each role includes people and groups that are assigned into.


  4. Let us know if you have any question or concern regarding Roles Management, thank you!
